# obj-inspector

Show all of properties in the Object.

## Installation

```
npm install --save-dev obj-inspector
```

## Example

```javascript
var inspector = require('obj-inspector');

var obj = {
  foo: "property1",
  bar: "property2"
};

inspector(obj);
```
